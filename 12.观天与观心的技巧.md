### 观天与观心的技巧

   前面讲“阴符”二字的意蕴之时，用了万余字进行阐述，上面讲这个“观”字，又花了万余字，但毕竟已经将一些原则性的东西都聊到了。下面继续看全经后面的文字。

   全文第二个字：天——古人经常说到“天”这个字，这个字往往因为语境的不同，有不同的意思。通常讲来，主要分为“形而上”与“形而下”两个方面。“形而上”的“天”，通常就是指“天道”，指“大道”。天道即大道。天道或者说大道，体现于自然之道，但并不一定完全是自然之道。“形而下”的“天”，就是我们抬头所看到的“天”。

   古人用形而下的“天”，来指代形而上的“天”，是因为两者有着共同的特点。那么共同在什么地方呢？在天上，我们往往看到云气的变幻，一会儿像棉花，一会儿像绵羊，一会儿像轻纱，一会儿像山峦；一会儿红，一会儿青，一会儿白……没有哪个时候会相同。云气虽然在不断变幻，但容纳这些云气的天本身，终不改动半分。云不断在变，就是“无常”，但不管云怎么变，天自身始终不会变，所以是“常”。如果问一下，天本身究竟是个什么样子？我们如何来回答？我们能回答像棉花？像轻纱？是红的？是白的？……显然不能。

   这里讲的是形而下的天，我们无法准确对它下定义。形而上的天，也是这样的。形而上的天，就是天道、大道，一切事物皆在天道、大道之中，就像所有的云都在天上一样。但是这个天道，我们无法说清楚它，就像我们无法给形而下的“天”下定义一样。所以《道德经》讲“道可道，非常道”——非常道，当然是无法真正说给它下定义，把它说清楚。

   佛家喜欢用云，或者用外界的境，来比喻人的“念”。然后以天，或者以镜子，来比喻人的“心”。我们心里，念头总是不断在变化，就像云或者像外境一样，是无常的；心，就像是天或者镜子一样，本身是不动不摇也无法下定义的。比如，“喜怒忧思悲恐惊”这“七情”，七种心理的状态，是“无常”的。可能这会儿喜，下会儿怒，再过会儿是恐……但在这个过程之中，包含有一个“常”——能“观照”到“喜怒忧思悲恐惊”的那个东西，一直在“经历着”种种“喜怒忧思悲恐惊”的那个东西，它本身，是不喜不怒不悲不恐……的，是“不动”的。所以，我们要把握住自己那个“不动”的东西，不要去追逐那些“无常”的，经常会变化的东西。这段话，放到前面“观”字中间去看，依然还是“观心”、“观念”。

   这种状态，禅宗大师船子德诚禅师《拨棹歌》中进行了描述：

```
   一片江云倏忽开，翳空朗日若为哉？
   适消散，又徘徊，试问本从何处来？
```

   云，是从哪里来？我们的七情，七种心理状态又从哪里来？是谁在“容纳”它们？这个在“容纳”的，本身并不是“七情”，我们能不能把它把握住？

   对内来说，情感、情绪、思维……等东西，是“无常”的，是在不断变化的，我们不去追逐、攀缘它，要谨守住“不变”的；对外来说，外界我们面临的种种环境、种种事件、种种机缘……，也是在不断变化的，也是“无常”的，“不变随缘，随缘不变”，我们同样需要守住自己的“常”，守住自己“不变”的那个，让它像镜子一样，照见种种变化，于种种变化之中，不动不摇。这点，无论面对内还是外，都是一样的。如果我们没有办法守住自己的“不变”与“常”，那么我们的心，就难免会随着种种不断变化的内外现象奔走、流转，用佛家的话来说，我们就仍然在“轮回”之中。

   事实上，一切思维、认识、观念、看法、情感、情绪、见地、理论……都不可能毫无缘由地自动就在我们内心中生成。它们的产生，势必会有外缘作为助缘，将其从我们内心中引发出来。例子也就是我们前文所举的，同一起车祸导致两兄弟完全不同的情感与认识、同一个马祖却导致四川人与江浙人完全不同的情感与观念、同一个中医也让人产生不同的感受与看法……这些看法、观念、情绪、情感，都是我们自己内在的问题，事实上跟外界助缘完全无关。助缘虽是外力，但种子来源于我们内在。但种子也罢，助缘也罢，都不是那个“常”的东西，种子、助缘以及两者相遇后生成的结果（观念情绪等）都是无常的。只有“平怀”是最接近于那个“常”的。如果我们的心识被种种无常的事物牵着鼻子奔走，那真的是非常的麻烦……

   一个人要在“无常”的种种现象之中作得主，那这个人的行履与见地，将是非常了不起的。那么，面对种种无常的，不断变化的心念、心态、心境、心情、心绪等等，我们究竟应该用什么样的心行或者说见地来处理之？南怀瑾先生用了如下一首诗偈来回答了这个问题：

```
   秋风落叶乱为堆，扫净复来千百回。
   一笑罢归闲处坐，任它着地自成灰。
```

   这首诗偈，为我们指明了“观心”时正确的态度——“任它着地自成灰”。任随念头生生灭灭，不有意去扫除它。不要老想着要“清静”，怕念头、杂念升起。我们应该让自己像是站在河边观水流一样，任念头流淌，只“观照”而不去管它，就行了。你如果起心要去“对付”它，或者“打扫”它，那么将永无穷尽之日——“扫净复来千百回”嘛。所以，对这个问题的“正见”，就是“不去管它”四个字而已。这个态度，也不仅仅只用于“观心”或者“观念头”，对待现实生活中的一些具体的事情，比如说有人在现实生活中给我们造成了麻烦，我们不妨也可以使用这个“不去管它”，用“不与它较量”的态度，来消除它。这个道理，如果用道家的术语来说，就是“夫为不争”，用禅宗的术语来说，就是前面我们已经说到过的“顺逆莫为”“不与一法为敌”。如果我们试图去扫除它，不管是试图扫除外境的还是内心的，那我们就已经与它为敌了，这东西就已经对我们造成影响了。再加上这些东西本来就是无常的，本身就是在不断变化的，我们一旦对其进行扫除，它就会变，我们不知不觉地，就会跟随它的变而进入“流转”，也就是佛家所说的轮回之中去了。所以，“任它着地自成灰”，是正确的见地，也是我们在对待自己内心与外界种种无常之事物时，应有的正确态度。“任它着地自成灰”也就是我们前文聊过的“泯然自尽”，等它自行消失。

   上面这段话，点明了“观心”与“应事”之时的一种“智慧”，这个智慧归纳起来，其实就是“不管它”三个字。事实上，观心，是向内；应事，是向外。看起来向内、向外有别，但从根本的智慧来讲，只有一个，那就是“不管”。前一节我们讨论过，历缘对境观，是定；因缘观，是慧；观心，包含了戒。为什么说“慧”的产生，是离不开“因缘观”的呢？我们不妨再看一个例子：昔日寒山问拾得曰：“世间有人谤我、欺我、辱我、笑我、轻我、贱我、恶我、骗我，如何处置乎？”拾得曰：“只是忍他、让他、由他、避他、耐他、敬他、不要理他，再待几年，你且看他。”这个例子中，拾得回答的核心，有两个，第一个，就是我们前面说的“不与它较量”，第二个，其实就是四个字——“再待几年”，这“再待几年”是时间上概念。我们在讨论因缘观的时候，也已经指出，因缘观是“线”状的，是从属于时间线上的一种“观”法，如果我们善于将自己面对的种种事件，放到“时间线”上去看，我们自然会对这种种的事件，有更清醒的认识，从而做出更好的应对——这本身，就是一种简单的智慧。

   前面已经说到，当我们面对内在或者外在的事物之时，可以用“不去管它”“不与它较量”的方式，来消除其对我们的影响。但是，无论是否管它、是否与其较量，这事物本身，毕竟是已经先存在了，才可能被我们“看”到。不去管它、不较量，是我们看到之后，所采取的应对方法。显然，如果我们根本就没有见到这事物，也就谈不上需要使用什么应对方法。这里，我们就可以引出学修之时另一个关键的要领，那就是三个字：“不顾视”。用道家的术语来说，就是“收视返听”，不必向外观照。这个要领，应该说更为根本，更为核心。我们现在还没有证道，还处于学修的过程之中，这个时候尽量减少外缘缠缚，对我们来说还是有一定意义的，而减少的方法，就是“不顾视”。如果自己已经完全对自己有了把握，已经既能入得了魔，又能入得了佛，能够任运无碍了，那顾不顾视，就全在自己了——旁人也难以知道你究竟是在佛地还是魔地——阴符了。

   闻听老辈人说起过的老传统中的一些“怪”规矩，比如有这么一项：谁要是看见扫帚倒了，自己“不声不响地”过去把它扶起来，就算了——这就是所谓的“见了便做了，做了便放下了”。这规矩好像没啥意思吧？别急，“意思”，在后面这点——如果，这位看见扫帚倒了的人，不是过去将扫帚扶起来，而是在那儿大叫“谁把扫帚弄倒了？！”那么，怪规矩就出现了——谁在大叫，那就是谁弄倒的！就必须受相应的惩罚！而且惩罚往往会有些让人难以接受。类似的例子还有一些，比如：谁看见没柴了，谁就得闭嘴上山去打柴！谁看见缸里没水了，谁就得闭嘴去挑水。总之，“闭嘴”然后“去做”，这两点是关键词。结果，在这样的“怪规矩”之下，很多人就养成了“视而不见”“听而不闻”的习惯。像这样有着“视而不见”“听而不闻”习惯的人，一放到花花世界中去，他眼中，恐怕确实就只剩下“天下太平”四个字了吧？！

   粗看之下，觉得这种规矩简直莫名其妙。但细细一想，却觉得大有道理——“视而不见”，未必“真的”眼没有看见，但“心”就确确实实没有看见；“听而不闻”，也未必是耳朵没有听见，但“心”就确确实实是没有听见……这种现象，前五识的感知功能与第六、七识之间，是“断开”的。按我自以为是的感觉，我觉得这怪规矩背后的要点，其实就是三个字：不顾视。一旦“顾视”，按那怪规矩，你看见了就得“闭嘴！去做！”，那就太复杂了。比如，看见这社会上的某些“不公”，看见野蛮执法，看见科技畸形发展对整个生态的危害……，既看到了，就必须做！那么，这要怎么去做？要知道，见了不做，就得挨板子啊！到底怎么办？！

   如果一个人在修为过程中，对“不顾视”有了感觉，他的心量与气质，也必将产生变化。聊到这儿，就又想起几句老话：

```
   饱谙世味慵开口，会尽人情只点头；
   如若连头也懒点，亦无烦恼亦无忧。
```

   这几句老话，看起来有些消极，但落实到我们自身的修养上来说，则确确实实有其积极的因素在内。如同练武术，从花招百出练到无招胜有招，只有达到无招的程度，才可能看透一切花招。只有“饱谙”与“会尽”世态，才可能真正在施为之时，做出既不会“过”也不会“不及”的、恰如其分的行动来。

   不能将自己修炼到“不顾视”的程度，那么我们的生存状态往往会出些意料之外的麻烦。简单举个例子，比如，现代社会网络发达，种种信息铺天盖地，如果我们做不到“不顾视”，对自己没有克制，则很可能陷入这样的信息之海中去。经常有人半夜三更还在上网，就是典型例子。这样做的后果，首先是损身害命，大违养生之道；另一个后果，则是自己被这种种信息给扰乱，或看到某些信息而产生情绪，或看到某些信息而形成某些乱七八糟的知见等等，这些后果，对自己影响更为深远，也更难以去除。总之，不要有好奇心，不要有盗心，不管自己现在是否做得到，但至少要有“不顾视”这样的心态。这，也是一种非常简单，但事实上并没有几个人能真正做到的“智慧”。

   不顾视，是我们对自己的心性进行调养之时应该注意的一个非常重要的原则。我们前面已经讲到了好几条重要原则，比如顺逆莫为、不与一法为敌等等。这些原则终归得落实到我们的心性与心行之中，才能够让我们真正地对自己的身心性命有所把握。不过，要落实到我们自己的心性与心行中，终归还是说起来容易，做起来难。我自己在这些方面有过很多的教训，偶尔想到，也深感一言难尽。对于一个走在学修道路上的人，我们最容易犯的错误，就是拿自己的观点与观念，去要求自己身边的人，特别是自己身边亲近的人。长期处于这样的状态之下，我们往往会将自己与自己周边的关系，搞得一团混乱。有时我想，我们学修，就如同森林里的树木一样，这棵树要想长成大树，唯一需要注意的，就是自己的眼睛一定要长在脑门顶上，不要去“顾视”自己身边其他的植物。我们瞄准自己的目标，一心向上，当自己确实已经长成大树之后，自然能给周边带来荫庇。在自己还未长成大树之前，顾虑这个，担心那个，像这样牵牵绊绊，总是难以放下心来一心向上，其最终的结果就是，自己也会像周边的植物一样，长成乱七八糟的小灌木。总之，在我们自己没有长成大树之前，我们一定要有凌云之心气，并且一定要注意不被周边环境过多的牵扯，当自己已经确有把握，确实能给周边带来荫庇之时，再来“关照”周边也不迟。“只管自己不管别人”在传统的禅堂中就是这样要求的。一大群禅僧在禅堂里，师父往往会提前教导大家进入禅堂之后的两句口诀“摩诃萨，不管他；阿弥陀，各顾各”！这两句口诀就是要大家将心思用在自己身上，这就是所谓的“自参自悟自成佛道”。三个“自”字，很说明问题。

   这一节我们本来是在解释“天”字，上面这些内容，我们既讨论了“观心”又讨论了“应事”。在前文很多地方我们也已经说到过——从更为根本的地方看，天与心、心与事，乃至于心、境、缘、事等等，皆是一体的东西。我们作为人而言，首先得把握好自己的心，才可能慢慢对天道产生感觉，同时，也才可能在应事之际，有更具智慧的处理方式。从这里我们也可以看出，心，确实就是“机”。

   于此，我们不妨再欣赏一首诗偈，作者是大名鼎鼎的济公——道济禅师。其诗偈意义与上述南怀瑾先生的诗偈类似。曰：
   
```
   着意求真真转远，痴心断妄妄犹多。
   游人一种平怀处，明月青山影在波。
```
