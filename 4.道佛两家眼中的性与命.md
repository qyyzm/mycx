### 道佛两家眼中的性与命

   事实上，打从唐、宋之后，道教的理论体系，已经明显有了佛化的倾向。比如后来道家所讲的“性命双修”，将炼内丹、内功，看成是“修命”，是对肉体的修炼；将“悟道”、参悟，看成是“练性”，是对心性的修炼。在相传为道家南、北两宗共同的祖师——吕洞宾，所作的《敲爻歌》中说：

```
   只修命，不养性，此是修行第一病；
   只悟祖性不修丹，万劫阴灵难成仙。
```

   这首诗偈，显然是将性、命二者对等来看的。

   道家所谓的“命”，主要就两个方面，一个，是练内丹，通俗的说法就是聚集生命的能量，其用来练习的基本对象，称之为“气”；另一个，是练内功，通俗的说法就是利用呼吸的力量，按摩、挤压、震荡、调理内脏，其用来练习的基本对象，称之为“息”，其实就是呼吸之“息”。平常我们往往将“气息”二字合称，这二字所指，也就是道家所说的“命功”。内丹与内功这两个方面，都是针对人的肉体在下功夫，其主要目的也就是为了调整人的物质肉体，让其达到一个非常纯粹、干净、健康的状态。

   道家所谓的“性”，其实是《道德经》《庄子》等经典所讲的，人的心性的修养与调节。其主要目的，当然是为了参透玄机，悟得“大道”，从而让自己的心志、认识、看法以及意识、思维方式包括情绪等等，处在一个较为理想与优化的状态之中。这种心理、情志、思维方式等等的理想与优化状态，道家称之为“逍遥”，佛家称之为“自在”或者“解脱”，其内涵实质上是一回事。

   那么，以佛家的眼光来看，命功与性功又是怎么一回事呢？

   命功的第一种，练内丹，看到自己肚子里或者脑袋中再或者胸腔里，有一坨光团，其实就是佛家《圆觉经》中所讲的“三摩钵提”，也是佛家天台宗“止观法门”中的一种“观法”。总而言之，这观，是一种“幻观”，是自己精神想像出来的东西。类似的，还有西藏密宗“大圆满”讲的观“明点”，不管是极大的“明点”还是极小的“明点”，不管是这样“灯明”还是那样“灯明”，其实跟这个内丹的道理一样，仍然是“幻观”。以佛家禅宗的话来说，搞这些事情，就是在“玩弄自己的精神”。真正的问题在于，既然这些光团、明点等等，都是精神或者意识“假想”而成的，那为什么在实际操作中会产生某种“效果”呢？这一点，才是真正的奥妙所在。历代之中，对这个核心奥妙，无论是道家还是佛家，都遮遮掩掩，不敢明说。后文我们讲到“心者，机也”的时候，如果机缘合适，或许可以聊聊这个话题。如果机缘不合适，因为某些特殊缘故，则我们也不好公开聊这个话题。如果有朋友确实有心想弄清这个问题，则建议通读佛家的《楞严经》。《楞严经》中非常巧妙地将这个问题分解在了整个所讲的经文之中，但仍然未能直接明言。南怀瑾先生所作《楞严大义》已经将这个奥妙提取了出来，不过依然没有明言。没有明言，其原因当然与道家佛家都不能明言，是同一个理由。在此，我们也只能点到为止——《楞严经》的全称是“大佛顶如来密因修证了义诸菩萨万行首楞严经”。在这个经文命名中，被称之为“诸佛密因”的那个东西，就是上述奥妙。这奥妙的最终体现就是：道家的内丹、佛家密宗的明点、天台宗的观想之类，虽然都是“幻观”，都是虚妄的，但如果连基本的效果都没有的话，则完全就是空谈，也就完全失去了存在的价值与必要。正由于其有效果，所以虽然是幻，但还是有其存在的理由。

　　关于《楞严经》中揭示的这个奥秘，宋代禅宗的法常禅师在临终之时，曾写下一首《渔父词》交待给自己的徒弟，他写道：
```　　
   此事《楞严》常露布，梅华雪月交光处；
   一笑寥寥空万古，风瓯语，迥然银汉横天宇。

   蝶梦《南华》方栩栩，斑斑谁跨丰干虎？
   而今忘却来时路，江山暮，天涯目送鸿飞去。
```

　　相传，法常禅师在入寂前的清晨，写下这首词即登榻敛足而逝。“此事楞严常露布”，当然是明确说，有这么个奥秘隐藏在《楞严经》中。不过，现在很多人都认为法常禅师词中所说的“此事”是指生与死。事实上，生死的奥秘与我们上面聊的修心、修性的奥秘，本质上是一回事。

   命功的第二种，练内功，把自己的呼吸调整来调整去，然后利用呼吸时腹内产生的压力，对内脏进行“按摩”、挤压、震荡与调整。这种方法与佛家的“安那般那”、“安般法门”，其实也是一回事。类似的，西藏密宗所修的“宝瓶气”、“九节佛风”同样是这么回事。换言之，无论是内丹还是内功，道家、佛家乃至于密宗，都是一样的，这些东西是大家都有的“共法”。这些共法真正的作用，就是要将我们的物质肉体，训练、调整到高度纯净、高度健康、高度健旺的程度。关于这命功的详细内容，我们后文讲“知之修炼，谓之圣人”以及“禽之制在气”等语句的时候，应该还会涉及，此处只是对其特点与作用简单介绍一下，后文我们再来详细讨论。

   说到“修性”，后代道家将“修性”变为“悟道”，是向佛家禅宗学习。因为从本质上讲，禅宗讲的“悟”、讲的“明心见性”，与《道德经》《庄子》等指出的人的心性修养，是没有根本区别的。这些地方，道佛两家理论体系虽然略有差异，其实所讲实质内容完全一致。向佛家的“参悟”学习，将道家的体系改向佛家靠拢，应该说，这就是张伯端在道家思想体系发展上的最大贡献。比张伯端略后的王重阳，创立道家全真教，性命并重，性命双修，也承袭了“道佛合一”的思想体系。

   如上所述，如果从更长远的历史时间来看，道家老子、庄子所流传下来的记述，其所讲，多半就是如何调整与优化人的心性、心智、思维方式与意识等。但后代道教过于重视“炼丹”、“练气”，过于重视物质肉体的修炼，而忽视了心性方面的修养，实际上已经离开了老庄时期的道家理论体系。吕洞宾、张伯端、王重阳等人，重新提倡心性的修养，就是为了挽救道家只重炼丹，只重“修命”的弊病。只不过，他们讲心性的修养，没有直接回复到老、庄的思想体系中去，而是顺手将佛家“参禅”“参悟”等东西拿过来用。从实质上看，佛家的思想特别是禅宗的精神，与老庄的体系并没有什么大的区别，所以，他们这样做，是完全没有问题的。

   那位以禅门宗匠自居，并写过颇具影响的《御制拣魔辨异录》的清世宗雍正皇帝，在评价张伯端的作品时说“《悟篇》不着宗门一语，《禅门歌》不着玄门一语，皆入理域，究明宗旨”。未将张伯端归之于“魔”与“异”，可见对张伯端的评价也是相当高的。

   张紫阳《禅宗歌颂诗曲杂言》中的《即心是佛颂》，被用在《西游记》第14回“心猿归正，六贼无踪”的卷首。当代禅宗大德、云门宗传人冯学成先生，在讲禅宗经典《信心铭》与《宝镜三昧》的时候，两次举到这首颂，认为就算是在禅宗之内，也是要相当有感觉的人才写得出：

```
   佛即心兮心即佛，心佛从来皆妄说；若知无佛亦无心，始是真如法身佛；
   法身佛，没模样，一颗圆光含万相；无体之体是真体，无相之相即实相；
   非色非空非不空，不动不静不来往；无异无同无有无，难取难舍难听忘；
   内外圆明到处通，一佛国在一沙中；一粒沙含大千界，一个身心万个同；
   知之须会无心法，不染不滞为净业；善恶万端无所为，便是南无及伽叶。
```

   为啥这里用了如此长的篇幅来讲张紫阳？因为作为道家南宗的祖师，对道，他当然有发言权，看他的《即心是佛颂》，说明他对佛、对禅，也有相当程度的体会，他大力推崇《阴符经》，自然有他独到的眼光。他是道家南宗的祖师爷，有他的推崇，《阴符经》在道家内的地位与重要性，自然是不言而喻的。

### 史上对《阴符经》的评价

   道家北宗“全真教”创始人王重阳也很重视《阴符经》。王重阳说自己“理透阴符三百字，搜通道德五千言”，其大弟子马丹阳也说“学道之人不须广看经书，若河上公注《道德经》、金陵子注《阴符经》，二者时看亦不妨”。丘处机的高足尹志平同样说过“道人虽未能广学，《阴符》、《道德》、《清静》三经，又岂可不学？”，可见道家北宗同样将《阴符经》看得很重。

   事实上不仅是道家，儒家历来也对《阴符经》十分重视。程颐说“《老子》言甚杂，如《阴符经》却不杂，然皆窥测天道之未尽者也”。朱熹化名“崆峒道士邹诉”，作《黄帝阴符经注解》，多处附加按语，赞誉其经文“极说得妙”、“最下得好”、“此等处特然好”。元代大儒刘因，也让其弟子读《阴符经》，他说“史既治，则读诸子者，庄列阴符四书者，皆出一律，虽云道家者流，其间有至理存”。明代哲学家吕坤评价说，《阴符经》其言“洞造化精微，极天人蕴奥，契性命归指。帝王得之以御世，老氏得之以养身，兵家得之以制胜，术数家得之以成变化而行鬼神，纵横家得之以股掌人群”……

   据此可以看出，《阴符经》自公布于众之后，即受到三教九流的关注，为众多思想家、学者所重视。有四句话评价《阴符经》，我觉得说得非常准确，那就是唐朝陆龟蒙对《阴符经》的评价：

```
   备识天地意，献词犯乾坤。
   口含造化斧，凿破机关门。
```

   总而言之，言而总之，《阴符经》“不测似阴阳，难名若神鬼”，是人类智慧的结晶。只要我们用心去品味它，去体会它，是能够以它为出发点，贯通于道、佛、儒三教的。当然，此处说贯通于道佛儒三教，也还是“小气”的说法，严格来说，以《阴符经》为机，贯通于一切“义理之学”，也未必是虚言。
